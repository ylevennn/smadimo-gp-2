# ГП2 - Инди-игры в Steam

### **Идея проекта**

В качестве идеи проекта был взят рынок видеоигр, в частности, представленных на площадке Steam. Нашим потенциальным заказчиком является небольшая студия разработки, которая хочет выпустить в Steam игру, при этом такую, которая с большей вероятностью получила бы больше охвата и положительных реакций от геймеров. Необходимо было изучить текущую конкурентную среду, попробовать найти такие факторы, которые положительно влияют на успех игры и предложить компании как наилучшие варианты для их продукта, так и наихудшие.

Было решено рассмотреть инди-игры.

> [Инди-игра](https://ru.wikipedia.org/wiki/Инди-игра) (англ. indie game, от англ. independent video game — «независимая компьютерная игра») — компьютерная игра, созданная отдельным разработчиком или небольшим коллективом без финансовой поддержки издателя компьютерных игр.

Выбор именно инди-игр был обоснован тем, что заказчик это не крупная корпорация с огромным бюджетом и ресурсами. Конкруентами являются как раз небольшие студии и одиночные разработчики, которые по определению выше и занимаются разработкой инди-игр.

При этом людям все чаще нравятся свежевыпущенные "простые" игры, которые доступны как можно большему числу игроков, которые становятся популярными на таких игровых платформах как Twitch, YouTube и TikTok (в частности нарезки со стримов), а не тяжелые AAA проекты, с завышенными ожиданиями и высокими требованиями к производительности и ценой.

Мы собрали датасет, применяя скрапинг/парсинг и API, подготовили его к работе и, проведя Exploratory Data Analysis (EDA) - сформировали итоговые выводы, которые будут презентованы заказчику.

### **Подготовка**

0. Склонировать репозиторий


1. Создать и активировать окружение

- На MacOS/Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

- На Windows (cmd):

```bash
python -m venv .venv
.venv\Scripts\activate
```

2. Установить требуемые библиотеки

```bash
pip3 install -r requirements.txt
```

#### Скрапинг:

- `src/scraping/collect_steam_data.py` - основной Selenium парсер, проходящий по списку со `steam_ids` и получающий следующую информацию:
  - `steam_id` - id игры в Steam, дублируется с indie*games | **тип признака:** *категориальный номинальный\_
  - `game_name` - название игры, как это написано в Steam | **тип признака:** _категориальный номинальный_
  - `game_description_snippet` - краткое описание игры | **тип признака:** _категориальный номинальный_
  - `game_price` - цена игры в рублях | **тип признака:** _количественный дискретный_
  - `found_game_price` - boolean поле, показывающее получили ли цену игры со страницы. Могут быть случаи нестандартной разметки страницы, где цену игры получить общим способом нельзя. | **тип признака:** _категориальный номинальный бинарный_
  - `all_language_reviews_type` - окраска обзоров на всех языка: Крайне положительные, В основном отрицательные и т.д. | **тип признака:** _категориальный порядковый_
  - `all_language_reviews_count` - количество обзоров на всех языках | **тип признака:** _количественный дискретный_
  - `has_russian_reviews` - boolean поле, показывающее - есть ли для игры обзоры на русском языке | **тип признака:** _категориальный номинальный бинарный_
  - `all_russian_reviews_type` - окраска обзоров на русском языке: Крайне положительные, В основном отрицательные и т.д. | **тип признака:** _категориальный порядковый_
  - `all_russian_reviews_count` - количество обзоров на русском языке | **тип признака:** _количественный дискретный_
  - `steam_app_url` - URL страницы с игрой в Steam | **тип признака:** _категориальный номинальный_
  - `support_ru_region` - boolean поле, показывающее доступна ли игра для российского региона. Определенный title страницы позволяет точно определить что по игре ничего не было получено именно из-за неподдерживаемого региона | **тип признака:** _категориальный номинальный бинарный_

#### API:

- `src/api/get_indie_games.py` - небольшой вспомогательный скрипт с запросом к публичному SteamSpy API для получения списка игр/steam_id с тегом Indie.
- `src/api/igdb.ipynb` - основной ноутбук для обогащения датасета с помощью API (с API ключом)
  Получаемые API-датасеты для обогащения:
- `data/igdb_steam_games.csv` — соответствие steam_id и igdb_id.
- `data/df_games.csv` — основные признаки по играм из IGDB:
  - `id` — id игры в IGDB | **тип признака:** _категориальный номинальный_
  - `name` — название игры | **тип признака:** _категориальный номинальный_
  - `rating` — средний пользовательский рейтинг по IGDB | **тип признака:** _количественный непрерывный_
  - `rating_count` — число пользовательских оценок по IGDB| **тип признака:** _количественный дискретный_
  - `total_rating` — средний рейтинг пользователей по IGDB и внешних оценок критиков | **тип признака:** _количественный непрерывный_
  - `total_rating_count` — число оценок пользователей по IGDB и внешних критиков | **тип признака:** _количественный дискретный_
  - `first_release_date` — дата первого релиза | **тип признака:** _дата/время_
- `data/df_popular.csv` — данные популярности:
  - `game_id` — id игры в IGDB
  - `external_popularity_source` — источник популярности (IGDB, Steam, Twitch)
  - `value` — значение метрики популярности
- `data/df_web.csv` — данные о внешних площадках/типах сайтов для игр.

#### EDA:

- `src/eda/merge_parsed_df.ipynb` - получение всех спаршенных датасетов, последовательное их склеивание и выгрузка готового `df_parsed` датасета, который далее будет обогащен при помощи API.

### **Используемый сервис для парсинга**

[Steam](https://store.steampowered.com/) - онлайн-сервис цифрового распространения компьютерных игр и программ, разработанный и поддерживаемый компанией Valve. Steam выполняет роль средства технической защиты авторских прав, платформы для многопользовательских игр и потокового вещания, а также социальной сети для игроков. Программный клиент Steam также обеспечивает установку и регулярное обновление игр, облачные сохранения игр, текстовую и голосовую связь между игроками. [Wikidepia](https://ru.wikipedia.org/wiki/Steam)

### **Дополнительный API сервис для подготовки нужной выборки игр**

Был использован сервис [SteamSpy](https://steamspy.com/) с публичным API, который одним [запросом](https://steamspy.com/api.php?request=tag&tag=Indie) позволил быстро получить id игр, относящихся к инди играм. Напрямую через Steam можно найти категорию с инди-играми, но получить весь список довольно проблематично + при случае, когда парсинг данных разделяется между командой для экономии времени - поэтому использование данного API было необходимым.

### **Основной используемый API сервис**

[IGDB.com](https://www.igdb.com/) is a video game community website, intended for both game consumers and video game professionals alike.

С данного сайта был взят IGDB API, где изучить документацию и найти инструкцию по получению нужных credentials можно [тут](https://api-docs.igdb.com/#getting-started)

#### Требуемые переменные окружения для API:

- `ACCESS_TOKEN` - получить через [запрос](https://id.twitch.tv/oauth2/token) после получения `CLIENT_ID` и `CLIENT_SECRET` через [Twitch Developer Portal](https://dev.twitch.tv/console/apps/create)

## Подведение итогов

### Выводы по результатам анализа

- Большинство игр оказалось с положительным откликом пользователей
- Распределения по отзывам и числу оценок имеют длинный хвост (много малых значений и немного очень больших) 
- Русскоязычный отзывы есть только у части игр, при этом как было выявлено в рамках анализа - все эти игры имеют неотрицательные отзывы по миру - has_russian_reviews=1 у 1414 игр из 8720
- Бесплатные игры имеют более высокую медиану отзывов: 1407 против 374
